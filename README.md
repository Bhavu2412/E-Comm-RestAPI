## Title 

<h1>E-Comm RestAPI</h1>

## Description

The E-Comm restAPI provides with the whole backend part of the E-Comm web present on the github.

## Features

- Display a list of products.
- Provides functionality to delete products.
- Provides a functionality to update products by the user who created it with role seller.
- Provides a functionality to create products by the user.
- Provides a functionality to get products.
- Login a user with data stored in the database.
- Signup for those who dont have an account
- Reset password functionality.
- User can add product to cart or remove from cart only if customer.
